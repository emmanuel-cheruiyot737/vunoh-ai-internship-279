# Vunoh Global Diaspora Assistant

## 📌 Project Overview
An AI-powered platform designed to solve the "informal channel" problem for Kenyans abroad. This assistant extracts intent, calculates risk, and generates multi-format communication for money transfers, service hiring, and document verification.

## 🛠 Tech Stack (Day 1 Selection)
- **Backend:** Django (Selected for its native ORM and internal use at Vunoh).
- **Frontend:** Vanilla JS, HTML, CSS.
- **AI Brain:** Gemini API / OpenAI (chosen for structured JSON output).
- **Database:** SQLite (for portability during the test).

## ⚖️ Risk Scoring Philosophy
My implementation focuses on three pillars of risk:
1. **Financial Velocity:** Urgent, high-value transfers.
2. **Real Estate Fraud:** Specifically targeting land title verification.
3. **Audit Trail:** Transitioning from "WhatsApp-only" to a structured task system.
